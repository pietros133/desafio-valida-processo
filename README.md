# Desafio ValidaProcesso

Projeto em Java que simula um processo seletivo, selecionando candidatos com base no salário pretendido e tentando contato via ligações simuladas.

## Como executar

1. Clone o repositório
```bash
git clone https://github.com/pietros133/desafio-valida-processo.git
```
2. Compile o código

```
 javac src/projetos/validaProcesso.java
```
3. Execute
```
 java -cp src projetes.ValidaProcesso
```
